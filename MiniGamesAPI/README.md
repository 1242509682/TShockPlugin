# MiniGamesAPI 豆沙小游戏API
- 作者: [Bean_Paste]https://gitee.com/Crafty/projects,羽学
- 出处: [bbstr]https://www.bbstr.net/r/107/
- MiniGamesAPI的开发初衷旨在为插件开发者提供快速的小游戏整体部署，此插件内置了许多小游戏插件常用到的类以及代码。

