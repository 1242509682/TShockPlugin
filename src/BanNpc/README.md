## 指令

| 语法         |    权限    |       说明       |
| ------------ | :--------: | :--------------: |
| /bm add [ID] | bannpc.use | 添加禁止生成 NPC |
| /bm del [ID] | bannpc.use | 移除禁止生成 NPC |
| /bm list     | bannpc.use |  查看禁止生成表  |
