# BanNpc 

- Authors: GK
- Source: GK

## Commands

| Command     | Permission |          Details          |
| ------------ | :--------: | :--------------: |
| /bm add [ID] | bannpc.use | Add NPC to block list |
| /bm del [ID] | bannpc.use | Remove NPC from Block List |
| /bm list     | bannpc.use |  View Block List  |

## Config
> Configuration file location：tshock/禁止怪物表.json
```json
{
  "阻止怪物生成表": [] //Block List
}
```
