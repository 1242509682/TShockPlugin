## 指令

| 语法           |        权限         |   说明   |
| -------------- | :-----------------: | :------: |
| "cbox", "领盒子" | itemBox.claim   | 领取盒子物品|
| "gbox", "给盒子" |  itemBox.give  | 赠送盒子到指定玩家|
| "rbox", "清盒子" |  itemBox.remove  | 清空所有盒子|
