# ChestRestore 无限宝箱和禁止箱子改名

- 作者: 佚名，Cjx修改，肝帝熙恩简单修改
- 无限宝箱插件，装上后全局宝箱无限拿取且不可更改，箱子名字必须有权限`chestopen.name`才可更改

## 指令

| 语法      |    权限     |        说明        |
| --------- | :---------: | :----------------: |
| 无 | chestopen.name |   有该权限的玩家才可改箱子名字   |
