# Maptp 双击大地图传送

- 作者: Nova4334，肝帝熙恩
- 出处: 未知
- 有权限的玩家打开大地图后，双击去往所指定的位置
- 更近一步是否允许传送到实心方块

## 指令

| Command |  Perminsion  |   说明   |
|---------|:------------:| :------: |
| None    | maptp.noclip | 	Allows double-click teleportation (excluding solid blocks)|
| None    |    maptp     | Allows double-click teleportation (including solid blocks)|
