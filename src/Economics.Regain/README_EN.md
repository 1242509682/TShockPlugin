##  directive

| Syntax | Privileges | Description |
| ------------------- | :--------------: | :------------: |
| /regain [quantity] | economics.regain | redeem handhelds |
| /regain list [page number] | economics.regain | View Redeemable Items |
| /regain | economics.regain | Redeeming Handheld Items |
