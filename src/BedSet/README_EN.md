# BedSet

- Author: pang2333
- Use the recall item to set the respawn point, and teleport there upon logout or death
  

## Command

| 语法           |        权限         |   说明   |
| -------------- | :-----------------: | :------: |
| /shome	     | bed.spawn.set       |  set spawn point|


## Config
> Configuration file location：tshock/Bed.json
```json
{
  "重生点": {} //Spawn Point
}
```
