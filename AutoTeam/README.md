# AutoTeamPlus 更好的自动队伍

- 作者: 十七，肝帝熙恩
- 出处: 无
- 自动分配一个组的玩家到特定队伍

## 更新日志

```
暂无
```

## 指令

| 语法           |        权限         |   说明   |
| -------------- | :-----------------: | :------: |
| /autoteam或/at | autoteam.toggle  |   开关自动队伍功能   |
| 无 | noautoteam  |   有该权限不会被自动分配队伍   |

## 配置

```json
{
  "组的队伍": {
    "组名字": "队伍名称中文或English",//本行和下面两行均可改
    "default": "red",
    "亲爱的": "红队"
  },
  "开启插件": true
}
```
