# Goodnight 宵禁

- 作者: Jonesn、羽学
- 出处: 无
- 每天指定时间禁止进服或禁止召唤怪物（满足在线人数自动解禁召唤怪物）

## 更新日志

```
v2.1.0
修复玩家加入服务器拦截方法
加入在线人数判断禁止召唤怪物
将配置项加以描述，并把禁怪物表整理为全进度BOSS的NpcID
修复每次/Reload都会写入一次内置怪物ID问题

v2.0.0
加入了禁止召唤怪物逻辑
羽学适配了.net6.0并重构了大部分方法
```

## 指令

| 语法                             | 别名  |       权限       |                   说明                   |
| -------------------------------- | :---: | :--------------: | :--------------------------------------: |
| /reload | 无 |  tshock.cfg.reload |    重载配置文件    |
