# Goodnight 宵禁

- 作者: Jonesn、羽学
- 出处: 无
- 每天指定时间禁止进服或禁止召唤怪物

## 更新日志

```
v2.0.0
加入了禁止召唤怪物逻辑
羽学适配了.net6.0并重构了大部分方法
```

## 指令

| 语法                             | 别名  |       权限       |                   说明                   |
| -------------------------------- | :---: | :--------------: | :--------------------------------------: |
| /reload | 无 |  tshock.cfg.reload |    重载配置文件    |
