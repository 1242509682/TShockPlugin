# Rainbow Chat 五彩斑斓聊天

- 作者: Professor X制作,nnt升级/汉化,肝帝熙恩、羽学更新1449
- 出处: 无
- 使玩家每次说话的颜色不一样.

## 更新日志

```
1.0.3
更新了渐变色值
```

## 指令

| 语法           |        权限         |   说明   |
| -------------- | :-----------------: | :------: |
| /rainbowchat 或 /rc | rainbowchat.use  |   查看菜单   |
| /rainbowchat 或 /rc 渐变| rainbowchat.use  |   开关彩虹聊天【渐变色】功能   |
| /rainbowchat 或 /rc 随机| rainbowchat.use  |   开关彩虹聊天【随机色】功能   |
