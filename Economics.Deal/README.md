## 指令

| 语法              |         权限         |       说明       |
| ----------------- | :------------------: | :--------------: |
| /deal push [价格] |  economics.deal.use  |   发布手持物品   |
| /deal list [页码] |  economics.deal.use  | 查看交易物品列表 |
| /deal buy [ID]    |  economics.deal.use  |   购买交易物品   |
| /deal recall [ID] |  economics.deal.use  |   撤回发布物品   |
| /deal reset       | economics.deal.reset |     重置交易     |

## 配置

{
  "最大显示页": 10,
