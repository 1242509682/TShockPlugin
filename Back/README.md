# Back 返回

- 作者: Megghy，肝帝熙恩
- 出处: github
- 可以返回上次玩家的死亡地点，可自定义冷却时间

```
暂无
```

## 指令

| 语法           |        权限         |   说明   |
| -------------- | :-----------------: | :------: |
| /back | back   | 返回死亡地点|

## 配置

```json
{
  "back冷却时间": 20
}
```
