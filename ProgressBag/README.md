# ProgressBag 进度礼包

- 作者: 少司命
- 出处: 无
- 每当完成一个进度时可以领取一个礼包

## 更新日志

```
暂无
```

## 指令

| 语法                  |   权限    |        说明        |
| --------------------- | :-------: | :----------------: |
| /礼包 领取 [礼包名称] |  bag.use  |      领取礼包      |
| /礼包 领取全部        |  bag.use  | 领取所有可领取礼包 |
| /礼包 list            |  bag.use  |    查看礼包列表    |
| /礼包 重置            | bag.admin |    重置领取礼包    |

## 具体进度请查看 ProgressType.cs 文件
