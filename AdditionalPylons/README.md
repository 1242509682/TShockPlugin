# AdditionalPylons 放置更多晶塔权

- 作者: Stealownz，肝帝熙恩优化1449
- 出处: github
- 晶塔环境不受限制，数量可提升
- 所有晶塔都无视环境
- 所有晶塔都至少可以放一个

## 更新日志

```
暂无
```

## 指令

| 语法           |        权限         |   说明   |
| -------------- | :-----------------: | :------: |
| 无 | AdditionalPylons   | 放置更多晶塔权限|
