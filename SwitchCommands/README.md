## 指令

| 语法           |        权限         |   说明   |
| -------------- | :-----------------: | :------: |
| /switch(开关) <添加/列表/删除/冷却/说明/权限忽略/取消/重绑/完成> （支持拼音） | switch.admin | 需要放置并点击1次开关才能使用该指令 |
