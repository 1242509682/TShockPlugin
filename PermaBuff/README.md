# PermaBuff 永久 buff

- 作者: 少司命
- 出处: 无
- 让玩家永久拥有 buff

## 更新日志

```
暂无
```

## 指令

| 语法                            |      权限      |                说明                 |
| ------------------------------- | :------------: | :---------------------------------: |
| /permabuff [buffID]             | permabuff.use  |   给予自身永久 buff(再次使用移除)   |
| /gpermabuff [玩家名称] [buffid] | gpermabuff.use | 给予玩家一个永久 buff(再次使用移除) |
| /clearbuffs                     | clearbuffs.use |          清空自身所有 buff          |

## 配置

```json
{
  "不可添加buff": []
}
```
