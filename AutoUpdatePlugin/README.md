# AutoUpdatePlugin 自动更新插件

- 作者: 少司命，Cai
- 出处: 本仓库
- 使用指令自动更新服务器的插件(仅本仓库)

## 更新日志

```
1. 更新指定插件命令
```

## 指令

| 语法           |        权限         |   说明   |
| -------------- | :-----------------: | :------: |
| /cplugin | AutoUpdatePlugin   | 检查插件更新|
| /uplugin | AutoUpdatePlugin   | 一键升级插件(需要重启服务器)|
| /uplugin -t [插件名] | AutoUpdatePlugin   | 更新指定插件|
