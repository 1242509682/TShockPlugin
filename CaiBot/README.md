# CaiBot适配插件

- 作者: Cai  
- 仓库: 此仓库
- Cai云黑机器人的适配插件127.0.0.1
- 使用 \-caidebug开启调试模式(机器人服务器会指向)

## 更新日志

```
v1.1 修复前置加载失败的问题
```

## 指令

```
暂无  
```

## 配置

```json
{
  "密钥": "", //由系统自动配置
  "白名单开关": false //Cai白名单的开关
}
```

----------
## 使用方法:

**1.将插件安装在ServerPlugins/  
2.重启服务器**
